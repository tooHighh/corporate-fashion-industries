# Corporate Fashion Industries

A modern, responsive landing page for a corporate fashion brand, built with **React**, **TypeScript**, and **Vite**.

---

## ✨ Features

- Modern hero section with bold brand messaging and visuals
- About Us, Projects, Sustainability, Stay Updated, and Get In Touch sections
- Responsive layout for desktop, tablet, and mobile
- Component-based structure with section-specific styles

---

## 🛠 Tech Stack

- **Framework**: React + TypeScript  
- **Build Tool**: Vite  
- **Styling**: CSS (section-based styles)  

---

## 📁 Project Structure (Overview)

- `index.html` – HTML entry  
- `src/main.tsx` – React entry  
- `src/App.tsx` – Root app component  
- `src/index.css` – Global styles  
- `src/assets/` – Images, video, and brand assets  
- `src/components/` – Section components:
  - `hero/` – Hero + navigation
  - `aboutus/` – About section
  - `projects/` – Projects/cards
  - `sustainability/` – Sustainability sections
  - `stayupdated/` – Newsletter section
  - `getInTouch/` – Contact section

## 🧩 Getting Started

```bash

cd "c:/Users/Marven/OneDrive/Desktop/react.JS/corporate-fashion-industry"

# install dependencies
npm install

# run dev server

npm run build

# preview production build
```

---

## 📜 Scripts

- `npm run dev` – Start Vite dev server  
- `npm run build` – Create production build in `dist/`  
- `npm run preview` – Preview the production build  
- `npm run lint` – Run linter (if configured)
